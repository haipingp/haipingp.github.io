二、安装与配置

  • YCM安装过程主要参考github上的官网地址就行

  • 不过，官网教程上主要讲了使用Vundle安装的过程，而目前Vundle安装YCM有问题，所以官网提供了一种使用vim原始安装插件的方法，不过还是可以使用其他的package manager进行安装：

    1. Clone the plugin as an optional package
    2. Build the extension (per README.md instructions).(这里如果只安装C相关语言的库，则只需要使用python3 install.py --clangd-completer，而不需要使用./install.py --all)

    4. To update, Pull latest, update submodules, and rebuild:

  • 由于YCM需要通过clang编译代码来实现，所以需要提供clang项目对应的编译选项，从而使YCM可以进行语义分析。可以使用两种方法生成YCM需要的编译选项:

    1. Use a compilation database
    2. Provide the flags manually
      这里我们使用第1种方法，其中提到了一种工具Bear，安装好Bear之后，使用bear -- make命令可以在当前项目目录下生成compile_commands.json，YCM可以通过该文件获取所需要的编译选项。
